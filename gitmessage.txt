# 안녕하세요? 이곳은 Vim 편집기입니다 :)
# 내용은 모두 지워주시고, 커밋 메세지만 남겨주시면 됩니다.

# 1. 편집 모드로 전환:
# Vim 편집기에서 i를 눌러 편집 모드로 전환합니다.

# 2. 커밋 메시지 작성: 
# 커밋 메시지를 원하는 양식에 맞게 작성합니다. 예를 들어, 다음과 같이 작성할 수 있습니다:
# ex 1)  [feature/add] 상품 검색 기능 구현
# ex 2)  [bugfix] 상품 장바구니에 추가되지 않는 버그 수정
# ex 3)  [refactoring] 사용자 인증 모듈 리팩토링

# 3. 커밋 메시지 작성 완료:
# 작성이 완료되면 <Esc> 키를 눌러 명령 모드로 전환합니다.

# 4. Vim 편집기 종료:
# 다음 중 하나의 방법으로 Vim 편집기를 종료합니다:
# - 커밋 메시지를 저장하고 Vim 편집기를 종료하려면 :wq (콜론과 w와 q를 연달아 입력) 를 누릅니다.
# - 커밋 메시지를 저장하지 않고 Vim 편집기를 종료하려면 :q! (콜론과 q와 느낌표를 연달아 입력) 를 누릅니다.
# 위의 단계를 따라 커밋 메시지를 작성하고 Vim 편집기를 종료하면 됩니다.
